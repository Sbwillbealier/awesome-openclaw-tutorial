## 🎛️ Control UI 全面升级（Dashboard v2）

### 模块化视图设计
- **全新布局**：采用模块化设计，将功能分为独立视图
- **五大核心视图**：
  - 概览视图（Overview）：系统状态总览
  - 聊天视图（Chat）：对话交互界面
  - 配置视图（Config）：系统设置管理
  - Agent 视图（Agents）：智能代理管理
  - 会话视图（Sessions）：历史会话记录

### 命令面板（Command Palette）
- **快速访问**：类似 VS Code 的命令面板
- **键盘快捷键**：快速调用所有功能
- **智能搜索**：模糊匹配命令名称
- **使用方法**：按 `Ctrl/Cmd + K` 打开

### 移动端优化
- **底部标签栏**：新增移动端导航，提升操作���验
- **响应式设计**：自动适配不同屏幕尺寸
- **触控优化**：更适合手机操作的界面设计

### 增强聊天工具
- **斜杠命令**：输入 `/` 快速调用命令
- **搜索功能**：在聊天记录中快速查找
- **导出功能**：导出聊天记录为多种格式
- **置顶消息**：重要消息置顶显示

## 🔒 安全增强（重要！）

### 设备配对安全
- **短期令牌**：`/pair` 和 `openclaw qr` 使用短期引导令牌
- **自动过期**：令牌时效性增强，降低安全风险
- **安全验证**：更严格的设备认证流程

### 插件信任机制
- **明确授权**：禁用隐式工作区插件自动加载
- **信任决定**：用户需要明确信任插件才能加载
- **安全审查**：增强插件安全性检查

### 命令权限控制
- **所有权验证**：`/config` 和 `/debug` 要求发送者所有权
- **权限隔离**：敏感操作需要更高权限

### 网关认证强化
- **令牌清理**：清除共享令牌 WebSocket 连接的未绑定客户端声明
- **连接安全**：增强 WebSocket 连接的安全性
