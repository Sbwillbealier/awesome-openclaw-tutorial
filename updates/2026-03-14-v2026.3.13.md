## 🧊 Docker 时区可控

### 新增环境变量
- **`OPENCLAW_TZ`**：设置容器时区
- **灵活配置**：支持任意时区设置
- **默认值**：使用 UTC 时区

### 配置示例
```yaml
# docker-compose.yml
services:
  openclaw:
    environment:
      - OPENCLAW_TZ=Asia/Shanghai
```

### 使用场景
- 🕐 **定时任务**：确保定时任务在正确时区执行
- 📅 **日志记录**：日志时间戳与本地时区一致
- 🌍 **多地区部署**：不同地区使用不同时区

## 🚀 升级指南

### 快速升级
```bash
# 1. 更新到最新版本
openclaw update

# 2. 重启服务
openclaw gateway restart

# 3. 验证版本
openclaw --version
# 输出应该是：openclaw v2026.3.13
```

### Docker 用户更新
```bash
# 1. 拉取最新镜像
docker pull openclaw/openclaw:v2026.3.13

# 2. 停止现有容器
docker stop openclaw
docker rm openclaw

# 3. 启动新容器（可配置时区）
docker run -d \
  --name openclaw \
  -e OPENCLAW_TZ=Asia/Shanghai \
  openclaw/openclaw:v2026.3.13
```
